# 大方針U: 特別イベントメンバー選択制

確定日: 2026-05-28

## 背景

特別イベント（BBQ等）はグループ全員が参加するわけではない。
全メンバー自動登録では不参加者が集計に含まれ、集金管理が不正確になる。

## 方針

- 特別イベント作成時は作成者のみが初期参加者
- 部長・権限者・フォーム内権限者がメンバーを個別に追加・削除
- 作成権限を部長のみ → 部長・権限者に拡張
- 支払い申告済み・承認済みメンバーは削除不可
- メンバー追加時に通知送信

## 依存前提

- P1: Supabase RLS（メンバー操作はサーバーサイドで権限チェック）
- 大方針K（特別イベント集計フォーム）の拡張

## 決定理由

任意参加イベントで不参加者を集計から除外し、正確な集金管理を実現するため。
